# Agenda

---

This is a course project for SYSU Software Engineering Training 2018. It can help you manage users and meetings.

## Usage

```
# 1. Get the code
git clone https://github.com/whichxjy/Agenda

# 2. Compile
cd Agenda
make

# 3. Run
make run
```
